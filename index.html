八木仁志 - ITエンジニア志望

自己紹介

40歳を迎えた頃、新しいことに挑戦したい気持ちが強くなり、FP2級の資格取得に挑戦。仕事と両立しながら学習し、合格した経験が大きな自信につながりました。

現在はその姿勢を活かし、PHP・JavaScript・SQLを中心にプログラミング学習を継続しています。年齢に関係なく挑戦し続け、ITの力で新しい価値を生み出せるエンジニアを目指しています。

これまでの仕事

印刷業の営業として約20年間勤務し、広報誌制作を中心に顧客管理・進行管理を担当。顧客課題のヒアリング力と調整力を活かし、300社以上を担当。Excel VBAやGoogle AppSheetで業務効率化を行い、年間120時間の作業時間削減を実現しました。

Excel VBAによる業務効率化

定型作業をVBAで自動化し、配送ミスゼロと年間120時間の工数削減を達成。業務全体の生産性向上に貢献しました。

Google AppSheetを活用したノーコード開発

在庫管理アプリをノーコードで開発。現在は限定的に実務利用中で、今後は使用用途を拡大し業務フロー全体の最適化にも貢献予定です。

学習中のスキル

鳥取県デジタル人材育成プログラム / WEBアプリ開発コースで以下を学習中：

  • Webシステム構築（例：SNSなど）
  • 設計〜開発・テストを通した実践的スキル習得
  • チーム開発での現場に近い体験

使用言語・ツール：HTML / CSS / JavaScript / PHP / Laravel / SQL / MySQL / Git / GitHub / Bootstrap / ChatGPT

プロジェクトの志望理由

印刷会社で20年間営業を経験する中で、ビジネス課題をITの力で解決し、価値を生み出せる人材になりたいと考えるようになりました。

本プログラムでプログラミングスキルを体系的に習得し、営業で培った顧客課題のヒアリング力と組み合わせることで、地域や企業のデジタル化推進に貢献できる人材を目指します。

将来の展望

地元企業のIT化や業務効率化を支援できるエンジニアとして成長し、地域社会に貢献できる存在を目指しています。
